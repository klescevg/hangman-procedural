# Hangman (procedural)

## Introduction
Welcome to the Hangman game project. This is a simplified version of the classic word-guessing game designed to run in a command-line interface. The goal of this project was to refresh coding skills and practice writing programs in a procedural style before moving on to OOP.

## Features
- Simple English-language interface
- Dictionary file contains only English words
- User input validation
- No visual representation of the hangman

## Requirements
- Java 17+

## How to run
1. Clone the repository:
```bash
git clone https://github.com/klescevg/hangman-procedural.git
```
2. Navigate to the project root directory:
```bash
cd hangman-procedural/
```
3. Compile the source code:
```bash
javac -d out src/com/klescevg/hangman/*.java
```
4. Run the main class:
```bash
java -cp out com.klescevg.hangman.Main
```
